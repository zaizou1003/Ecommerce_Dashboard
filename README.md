# E-commerce Dashboard

## Project Overview
This repository contains a Power BI dashboard designed to analyze and visualize e-commerce data. The primary objective of this dashboard is to provide actionable insights into customer behavior, sales performance, and delivery trends.

## Files in the Repository
1. **Ecommerce_Dashboard.pbix**: The Power BI dashboard file containing all visualizations and data insights.
2. **Dataset_Ecommerce.xlsx**: The dataset containing e-commerce transaction details.

## Dataset Details
The dataset used in this project contains the following columns:

### `Dataset_Ecommerce.xlsx`
- **OrderDate**: The date when the order was placed.
- **OrderID**: Unique identifier for each order.
- **Delivery Date**: The date when the order was delivered.
- **CustomerID**: Unique identifier for each customer.
- **Customer Age**: Age of the customer.
- **Customer Gender**: Gender of the customer.
- **Location**: Location of the customer.
- **Zone**: Zone or region of the customer.
- **Delivery Type**: Type of delivery chosen (e.g., Standard, Express).
- **Product Category**: Broad category of the product (e.g., Electronics, Clothing).
- **SubCategory**: Subcategory of the product.
- **Product**: Name of the product.
- **Unit Price**: Price per unit of the product.
- **Shipping Fee**: Fee charged for shipping.
- **Order Quantity**: Quantity of the product ordered.
- **Sale Price**: Final price of the order after discounts and fees.
- **Status**: Status of the order (e.g., Delivered, Canceled).
- **Reason**: Reason for order cancellation (if applicable).
- **Rating**: Customer rating of the product or service.

## Dashboard Features
1. **Sales Analysis**:
   - Total sales and revenue trends over time.
   - Analysis of order quantities and unit prices.

2. **Customer Insights**:
   - Distribution of customer age and gender.
   - Location-based customer segmentation.

3. **Product Performance**:
   - Best-performing product categories and subcategories.
   - Rating distribution for products and services.

4. **Delivery Insights**:
   - Delivery time analysis.
   - Trends based on delivery type.

5. **Order Status**:
   - Proportion of delivered vs. canceled orders.
   - Analysis of reasons for cancellation.

## How to Use
1. Download or clone this repository.
2. Open the **Ecommerce_Dashboard.pbix** file in Power BI Desktop.
3. Load the `Dataset_Ecommerce.xlsx` dataset to refresh the data.
4. Explore the dashboard for insights.
